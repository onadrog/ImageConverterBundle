# ImageConverterBundle

Convert your images into WebP format.

## WIP

### Requirements
- PHP >= 8.0
- [GDImage](https://www.php.net/manual/en/intro.image.php) (outside docker image)
- [docker](https://docs.docker.com/)

MakeFile:

```bash
#run tests
Make tests

#run phpstan
make phpstan

#run console command
make console
(e.g: make console cache:clear)
```
